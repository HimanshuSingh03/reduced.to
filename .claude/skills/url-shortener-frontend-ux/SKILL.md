# Skill: url-shortener-frontend-ux

## Purpose

Guide UI/UX changes for the Reduced.to frontend.

## Rules of Thumb

- Show slug previews clearly.
- Support copy-to-clipboard actions for short links.
- Follow the existing layout and component structure.
